# MOMENT 5 - WEBBPLATS
Syftet med webbtjänsten är att:

* skapa en REST-baserads webbtjänst
* bekanta sig med relevanta begrepp & förstå hur att använda dessa
* använda Fetch-API i att konsumera en webbtjänst
* skapa webbplats med HTML/CSS/JS som konsumerar webbtjänsten

## WEBBSIDAN
För att lägga in kurser, skriv in i formulär. Tryck på knappen. Ta bort genom att trycka på någon kursnamn. Kursplan finns som länk till höger om kursinformation.
